# imghome
图床

**请求URL：** 
- ` http://bootzhong.top/imghome/upload`
  
**请求方式：**
- POST 

 **返回示例**

``` 
{
    "status": 0,
    "data": {
        "name": "f9f99547b5644eb69474861eb67a0602.png",
        "url": "http://bootzhong.top/img-home/f9f99547b5644eb69474861eb67a0602.png"
    },
    "msg": null
}

status : 0成功， 其他失败
url : 图片的url
```
